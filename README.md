# Golang DI Sample

## How to run the api
* `docker-compose up -d`
## API list
* User signup 
    * Method: POST
    * URL: /api/users
* User login
    * Method: POST
    * URL: /api/users/login
* Create article 
    * Method: POST
    * URL: /api/articles    
* Get articles 
    * Method: GET
    * URL: /api/articles
* Like article 
    * Method: PATCH
    * URL: /api/articles/:id/like
